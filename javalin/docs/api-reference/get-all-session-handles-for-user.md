# `getAllSessionHandlesForUser(String userId)`
### Parameters
- `userId`
    - **description:** All sessions of the give userId are revoked.

### Returns
- `String[]`
    - Each element in the `String[]` is a session handle

### Throws
- **[GeneralException](./error-handling/general-error)**
