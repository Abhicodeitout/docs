# ``UNKNOWN_EMAIL_ERROR``
**Type** : ``{type: UNKNOWN_EMAIL_ERROR, message: string}``
- The ``message`` string inside contains the error message
- The ``UNKOWN_EMAIL_ERROR`` is thrown when an unknown ``email`` is used to access an API.
