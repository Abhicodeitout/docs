# ``signUp(email, password)``

### Parameters
- ``email``
  - type: ``string``
- ``password``
  - type: ``string``

### Returns
- ``Promise<User>`` on successfull sign up. To know about the ``User`` object click [here](https://github.com/supertokens/core-driver-interface/wiki#user)

### Throws
- If the email used already exists.
