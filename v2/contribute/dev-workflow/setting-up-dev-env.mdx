# Setting up dev env

:::important
At the moment, our dev env only works on Mac or Linux.
:::

In general, you should follow the `CONTRIBUTING.md` file of the repo that you want to work with. At the moment, some of the setup is slightly more complex than it should be because you have to also setup other parts of the SuperTokens stack.
