# Clear cookies (if present) when clearing session in header mode

## Context and Problem Statement

We can choose to clear or leave a cookie based session if present when logging out using header based auth

## Considered Options

* Leave
* Clear

## Decision Outcome

Clear, because othewise a session might get stuck (or 2 sessions might be present) after logout.
