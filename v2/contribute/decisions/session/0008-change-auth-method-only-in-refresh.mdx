# Change auth method only in refresh calls

<DecisionHeader status="proposed" lastUpdate="2022-10-25" created="2022-10-25" deciders={["rishabhpoddar", "porcellus"]} proposedBy={["porcellus"]} />

## Context and Problem Statement

We need to decide if we can upgrade in get/verifySession on refresh calls

## Considered Options

* All
* Refresh only

## Decision Outcome

Refresh only, because this changes both the access and refresh tokens.
- Make sure that updating the access token updates in the fallback case use the same as the incoming request
