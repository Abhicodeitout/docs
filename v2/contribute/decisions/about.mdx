This is where we store a log of our more complex/important decisions. Please not that this is not an exhaustive log, it was started well after the project.

To propose a new decision:
1. Make a copy of `00XX-template.mdx` in the appropriate folder inside decisions and edit it.
2. Add the new decision to the sidebar in `sidebars.js`
3. Create a PR
