# Updating the JWT

In order to update the JWT, you first need to fetch session information. You can do this in the following ways:
- [Using verifySession on your APIs](../session-verification-in-api/verify-session)
- [Using session handles](../fetch-sessions-for-user)

<BackendSDKTabs>
<TabItem value="nodejs">

```jsx
// Get session information using one of the methods listed above
let sessionInformation = getSessionInformation();

let newClaims = {
    role: "admin",
}

await sessionInformation.updateAccessTokenPayload(newClaims);
```
