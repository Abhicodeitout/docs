# Disabling APIs

To disable an API entirely, all you need to do is override the api implementation with `undefined`.

For example, if you want to disable sign-up / sign-in api from this recipe, all you do is this:

<FrontendSDKTabs>
<TabItem value="reactjs">

```jsx
SuperTokens.init({
    appInfo: {...},
    supertokens: {...},
    recipeList: [
        EmailPassword.init({
            // highlight-start
            override: {
                apis: (originalImplementation) => {
                    return {
                        ...originalImplementation,
                        refreshPOST: undefined
                    }
                }
            }
            // highlight-end
        })
    ]
});
```
</TabItem>
</FrontendSDKTabs>

:::important
You then need to define your own routes that will handle this API call. You can see the [Frontend driver interface API spec here](../../apis)
:::
