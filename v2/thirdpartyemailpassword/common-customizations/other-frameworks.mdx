If your backend framework is not supported by SuperTokens, you can follow this guide on setting up an authentication server to protect your frontend and backend.

To illustrate how this would work, lets take an example:
- The user has a React based frontend and a php backend server.
- SuperTokens does not have a php SDK, so we will have to spin up a separate service to act as an authentication server.
- This authentication server can be written in NodeJS, Python or Golang. For this example we will be using NodeJS.
