# Sign out

<FrontendSDKTabs>
<TabItem value="reactjs">

The [`signOut` method](https://supertokens.io/docs/auth-react/modules/recipe_^{docsLinkRecipeName}.html#signOut) revokes the session on the frontend and backend.

```jsx
// highlight-next-line
import { signOut } from "supertokens-auth-react/recipe/^{codeImportRecipeName}";

function NavBar () {
  async function onLogout () {
      // highlight-next-line
      await signOut();
      window.location.href = "/";
  }
  return (
    <ul>
         <li>Home</li>
          // highlight-next-line
          <li onClick={onLogout}>Logout</li>

    </ul>
  )
}
```
</TabItem>

<TabItem value="vanillajs">

The [`signOut` method](https://supertokens.io/docs/website/modules.html#signOut) revokes the session for the user.

```jsx
// highlight-next-line
import SuperTokens from "supertokens-website";

async function logout () {
  // highlight-next-line
  await SuperTokens.signOut(); 
  window.location.href = "/";
}
```

</TabItem>
</FrontendSDKTabs>

- We do not provide any UI for a Sign-out button
- On success, the `signOut` function does not redirect the user to another page.
