# Get User Info on the Backend

You can a user's information on the backend using the `getUserByEmail` and `getUserById` functions:

## Using `getUserByEmail`

<BackendSDKTabs>
<TabItem value="nodejs">

```jsx
app.get("/get-user-info", async (req, res) => {

    let userInfo = await ^{recipeNameCapitalLetters}.getUserByEmail("test@example.com")
    res.send({
        userInfo
    })
})
```

</TabItem>
</BackendSDKTabs>

## Using `getUserById`

<BackendSDKTabs>
<TabItem value="nodejs">

```jsx
app.get("/get-user-info", Session.verifySession(), async (req, res) => {

    let userId = req.session.userId;
    let userInfo = await ^{recipeNameCapitalLetters}.getUserById(userId)

    res.send({
        userInfo
    })
})
```

</TabItem>
</BackendSDKTabs>
