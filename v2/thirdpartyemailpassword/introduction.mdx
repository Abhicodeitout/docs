# About this recipe
## Use this recipe to implement the following functionality in your applications:
- ✅ Sign-up / Sign-in with email ID and password
- ✅ Sign-up / Sign-in with third party providers (Like google, facebook etc..)
- ✅ Forgot password flow using email
- ✅ Secure session management
- ✅ Email verification

## [View the demo app](https://thirdpartyemailpassword.demo.supertokens.io/)
## [View the source code for the demo app](https://github.com/supertokens/supertokens-demo-react/tree/thirdpartyemailpassword)
