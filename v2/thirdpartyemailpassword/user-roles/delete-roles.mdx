# Deleting roles

You can delete any role you have created, if the role you are trying to delete does not exist then this is a no-op.

<BackendSDKTabs>
<TabItem value="nodejs">

```tsx
import UserRoles from "supertokens-node/recipe/userroles";

async function getAllRoles() {
    // highlight-start
    // Delete the user role
    const response = await UserRoles.deleteRole("user");

    if (response.didRoleExist === false) {
        // There was no such role
    }
    // highlight-end
}
```
