# Making Requests

:::info
We use Apollo client as an example, SuperTokens can be used with other GraphQL clients.
:::

```jsx
import { gql, useQuery } from '@apollo/client';

const HELLO_WORLD = gql`
  query {
    hello
  }
`;

function HelloWorld() {
    const { loading, error, data } = useQuery(GET_DOGS);

    if (loading) return 'Loading...';
    if (error) return `Error! ${error.message}`;

    return (
        <div>
            {data.hello}
        </div>
    );
}
```

Refer to the [official guide](https://www.apollographql.com/docs/react/data/queries/) to know more about queries.
