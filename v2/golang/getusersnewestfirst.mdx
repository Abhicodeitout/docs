## `supertokens.GetUsersNewestFirst(paginationToken *string, limit *int, includeRecipeIds *[]string)`

### Parameters
- ``paginationToken`` (Optional)
  - Token used to fetch the next set of users.
  - Pass `nil` to get the first page of users.
- ``limit`` (Optional)
  - Limit the number of users returned.
  - This is 100 by default
- ``includeRecipeIds`` (Optional)
  - Only return users who have signed up with that recipe.
  - `nil` by default (Gets all users)


### Returns
- `(supertokens.UserPaginationResult, error)`
  ```go
  type UserPaginationResult struct {
    Users []struct {
      RecipeId string
      User     map[string]interface{}
    }
    NextPaginationToken *string
  }
  ```

### Additional Information:
- If the ``NextPaginationToken`` is `nil`, then there are no more users to loop through.
- If there are no users in your app, then ``NextPaginationToken`` will be ``nil`` and `Users` will be an empty array
