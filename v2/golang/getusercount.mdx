## `supertokens.GetUserCount(includeRecipeIds *[]string)`

### Parameters
- `includeRecipeIds`
    - If provided, it will return users who signed up using those recipe IDs mentioned in the array.
    - Pass `nil` to get all user counts

### Returns
- `(float64, error)` number of users in your app or an error
