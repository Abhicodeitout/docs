# Frontend Customisation

If you are building your own UI, then you should use our supertokens-website repository for session management, and query the APIs exposed by the backend SDK yourself using the [FDI APIs](https://app.swaggerhub.com/apis/supertokens/FDI).

If instead, you are using our supertokens-auth-react SDK, then, [here is the list of customisations you need to make](https://github.com/supertokens/supertokens-auth-react/tree/master/examples/with-phone-password#frontend).
