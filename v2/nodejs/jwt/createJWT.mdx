## `createJWT(payload, validity)`

### Parameters
- `payload`
  - type: `any`
  - Must be a valid JSON

- `validity` (Optional)
  - type: `number`
  - Must be greater than 0

### Returns
```ts
Promise<{
    status: "OK",
    jwt: string,
} | {
    status: "UNSUPPORTED_ALGORITHM_ERROR",
}>
```
