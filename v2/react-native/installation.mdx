# React Native SDK

:::important
This documetation is only for reference, and not a step by step guide for how to implement SuperTokens.

For a guide, please see the [Getting started section](/docs/community/recipes).
:::

## Installation

```bash
npm i --save supertokens-react-native@^2.0.0
# IMPORTANT: If you already have @react-native-async-storage/async-storage as a dependency, make sure the version is 1.12.1 or higher
npm i --save @react-native-async-storage/async-storage
```
