# Changing the Magic Link URL

You can change the URL of Magic Links by providing a callback on the backend.

<BackendSDKTabs>
<TabItem value="nodejs">

```jsx
SuperTokens.init({
    appInfo: {...},
    supertokens: {...},
    recipeList: [
        Passwordless.init({
            ...
            // highlight-start
            // Customize information in the URL.
            // By default: `${websiteDomain}/auth/verify`
            // `?rid=passwordless&preAuthSessionId=${preAuthSessionId}#${linkCode}` will be added after it.
            getLinkDomainAndPath: async (contactInfo, context) => {
                return "https://example.com/custom/auth/verify"
            }
            // highlight-end
        }),
        Session.init({...})
    ]
});
```
</TabItem>
