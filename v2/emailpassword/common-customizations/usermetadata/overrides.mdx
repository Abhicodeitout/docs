# Overriding functions

You can override all functions provided by the recipe. Although these are only called by your implementation, you can use these to implement custom behavior in a single location.

## Use the override config

<BackendSDKTabs>
<TabItem value="nodejs">

:::info
See all the [functions that can be overridden here](https://supertokens.com/docs/nodejs/modules/recipe_usermetadata.html#RecipeInterface)
:::

```tsx
import SuperTokens from "supertokens-node";
import UserMetadata from "supertokens-node/recipe/usermetadata";

SuperTokens.init({
    appInfo: {
        apiDomain: "...",
        appName: "...",
        websiteDomain: "..."
    },
    supertokens: {
        connectionURI: "...",
    },
    recipeList: [
        UserMetadata.init({
            // highlight-start
            override: {
                functions: (originalImplementation) => {
                    return {
                        ...originalImplementation,

                        // here we are only overriding the function that's responsible
                        // loading the metadata object
                        getUserMetadata: async function (input) {
                            // TODO: some custom logic

                            // You can call the default behaviour as shown below, but this isn't strictly necessary
                            const res = await originalImplementation.getUserMetadata(input);

                            // You can then extend (or replace) the result of the original implementation with your own
                            return {
                              ...res,
                              metadata: {
                                ...res.metadata,
                                role: "admin",
                              }
                            }
                        },
                        // ...
                        // TODO: override more functions
                    }
                }
            }
            // highlight-end
        })
    ]
});
```

- `originalImplementation` is an object that contain functions that have the original implementation for this recipe. It can be used in your functions as a way to use the SuperTokens' default behaviour.
