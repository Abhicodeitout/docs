# Embed in a page 

## Step 1: Disable the default implementation 

<FrontendSDKTabs>
<TabItem value="reactjs">

```jsx
SuperTokens.init({
    appInfo: {...},
    recipeList: [
        EmailPassword.init({
            // highlight-start
            emailVerificationFeature: {
                mode: "REQUIRED",
                disableDefaultImplementation: true
            },
            // highlight-end
        }),
    ]
});
```
</TabItem>
</FrontendSDKTabs>

If you navigate to `/auth/verify-email`, you should not see the widget anymore.


## Step 2: Render the component yourself 

Add the `EmailVerification` component in your app:

<FrontendSDKTabs>
<TabItem value="reactjs">

```jsx
// highlight-next-line
import {EmailVerification} from 'supertokens-auth-react/recipe/emailpassword';

class EmailVerificationPage extends React.Component {
    render() {
        return (
            <div>
                // highlight-next-line
                <EmailVerification/>
            </div>
        )
    }
}

```
</TabItem>
</FrontendSDKTabs>
