# Getting public keys for JWT verification

SuperTokens exposes a Json Web Key Set that can be used for JWT verification, refer to [this page](./get-jwks-and-issuer) to know how to get the URL for this endpoint.

A sample response is:

```json
{
    "keys": [
        {
            "kty": "string",
            "kid": "string",
            "n": "string",
            "e": "string",
            "alg": "string",
            "use": "string",
        }
    ],
}
```

:::info
To use the certificate string directly, you can use this endpoint and then convert the JWK to a PEM string.
:::
