# Session Object
A `Session` object is returned when a session is verified successfully. Following are the functions you can use on this `session` object:
- [`get_user_id`](./get-user-id.md)
- [`get_session_data`](./get-session-data.md)
- [`update_session_data`](./update-session-data.md)
- [`get_jwt_ayload`](./get-jwt-payload.md)
- [`update_jwt_payload`](./update-jwt-payload.md)
- [`revoke_session`](./revoke-session.md)
