# Session Object
A `Session` object is returned when a session is verified successfully. Following are the functions you can use on this `session` object:
- [`GetUserID`](./get-user-id.md)
- [`GetSessionData`](./get-session-data.md)
- [`UpdateSessionData`](./update-session-data.md)
- [`GetJWTPayload`](./get-jwt-payload.md)
- [`UpdateJWTPayload`](./update-jwt-payload.md)
- [`RevokeSession`](./revoke-session.md)
